1. При создании параметров для функции check_access обзятельно нужно добавлять: groups => ['name_group']. При этом группа должна быть одна и совпадать с названием модуля.

2. При добавлении новой роли, открывать файл validator.php, находить метод check_access и добавлять в массив $roles, например 'head' => 5. 'head' в данном случае это название роли на английском языке. Соответсвенно теперь данная роль доступна при создании прав. Т.е. можно в контроллере написать:

	$roles_groups = [
		'roles' => ['head'],
		'groups' => ['tesla']
	];
	$this->check_access($roles_groups);

Так же открывать файл UserController.php и в функцию get_condition() дописывать вывод новой роли
